Pengertian Eigenface

Eigenface adalah metode pengenalan wajah (face recognition) yang menggunakan konsep aljabar linear untuk menemukan pola penting pada wajah manusia.

Metode ini menggunakan Singular Value Decomposition (SVD) atau Principal Component Analysis (PCA) untuk mengurangi dimensi data citra wajah.

Tujuan Eigenface

  • Mengenali wajah manusia
  • Mengurangi ukuran data gambar
  • Mencari ciri khas wajah
  • Membantu sistem face recognition

Langkah Algoritma Eigenface

  1. Membaca dataset gambar wajah
  2. Mengubah gambar menjadi grayscale
  3. Resize gambar agar ukurannya sama
  4. Mengubah gambar menjadi vector
  5. Menghitung mean face
  6. Normalisasi data wajah
  7. Menggunakan SVD
  8. Menghasilkan Eigenface

Rumus Eigenface Menggunakan SVD

A = X - μ

Digunakan untuk normalisasi data wajah.

A = UΣVᵀ

Singular Value Decomposition (SVD) digunakan untuk mendapatkan eigenface.

Code Python Eigenface

import os
import cv2
import numpy as np
import matplotlib.pyplot as plt

path_dataset = "datawajah"

images = []
image_shape = None

for filename in os.listdir(path_dataset):

    img_path = os.path.join(path_dataset, filename)

    img = cv2.imread(img_path, cv2.IMREAD_GRAYSCALE)

    img = cv2.resize(img, (100, 100))

    image_shape = img.shape

    img_vector = img.flatten()

    images.append(img_vector)

X = np.array(images)

mean_face = np.mean(X, axis=0)

A = X - mean_face

U, S, VT = np.linalg.svd(A, full_matrices=False)

num_eigenfaces = min(5, len(images))


eigenfaces = VT[:num_eigenfaces]

fig, axes = plt.subplots(1, num_eigenfaces, figsize=(15, 5))

for i in range(num_eigenfaces):

    eigenface = eigenfaces[i].reshape(image_shape)

    axes[i].imshow(eigenface, cmap='gray')
    axes[i].set_title(f"Eigenface {i+1}")
    axes[i].axis("off")

plt.show()
